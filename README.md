##3630海钓俱乐部挑战赛DEOM

演示地址：http://hd.fami2x.com

思路整理：
1、“海钓在欧美发达国家已有上百年的历史，高尔夫、马术和网球被列入四大贵族运动之一。”
既然和高尔夫类似，就以这个为出发点去设计，从最常规的角度可以分析海钓有一些特性标签：
A 贵，每次海钓出海需要各种设备还要租船，钓上来的鱼还需要专业保存。
B 协作，在海上相对封闭一般不可能会独自出海应该。虽然作为一个俱乐部的活动，应该需要体力、需要配合，可以折射出社会人与人之间的关系。同时，这种长时间的共同协作和封闭性会造成小圈子的强烈自豪感。
C 珍惜食材，是每一个大自然赐予每一个参与海钓者的证明，每一条鱼可能代表着在整个活动中的故事，代表着每一个人。
从这几个特性来说，准备从鱼和获取鱼的人入手，打造一款更贴近社交活动的海钓平台。

2、在整个项目中很多地方使用了专属顾问的方式，体现服务的定制性、尊贵性。
3、整体的基本功能为预约上门金枪鱼或其他鱼类的解体秀，在这个过程中渗透了一些小应用去扩展：
A 可在线定制菜谱 （衍生线下餐饮行业）
B 可在线定制增值服务：红酒／雪茄.... (衍生高端消费品市场)
C 仅自己钓上来的鱼可享受此服务，可通过平台预约出海航线。（VIP会员体系）
D 其他人也可以通过其它方式获取这些食材：
向其它会员借：通过微信扫描二维码转让金枪鱼的所有权。
将金枪鱼拍卖：自主定价时间拍卖（暗标）
将金枪鱼众筹：以名人晚宴的形式众筹与大牛们共同享用珍贵食材。

总体上从以上方面考虑，开发了这个小DEMO。
